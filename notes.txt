>>> First a user connects to a server by using username and password, it logs onto its account and display its information. It shows the list of previous chats and in those chats, it displays the previous messages. When opening a chat, the input is bound to the username of the contact. When message is sent, it sends the message with the username and message to the server. The server matches the username to its database of logged users and if found, it sends the message to the user. The second user when logs in, it sees the message.
